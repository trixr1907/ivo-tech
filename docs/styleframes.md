# Styleframe Gate

Drei schnelle Styleframes fuer die visuelle Richtungsentscheidung vor weiterem UI-Feintuning:

- `https://ivo-tech.com/styleframes/serious-modern.html`
- `https://ivo-tech.com/styleframes/bold-creative.html`
- `https://ivo-tech.com/styleframes/current-cleaner.html`

Lokal pruefen:

```bash
npm run dev
# dann im Browser oeffnen:
# /styleframes/serious-modern.html
# /styleframes/bold-creative.html
# /styleframes/current-cleaner.html
```
