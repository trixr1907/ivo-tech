# Homepage Agent Operating Rules

1. Always retrieve before judging.
2. Attach source IDs to every important claim.
3. Explicitly mark contradictions.
4. Label weak evidence as low confidence.
5. Avoid generic recommendations without project-specific evidence.
6. Separate three layers in every output:
   - current-state fact
   - benchmark/inspiration pattern
   - final recommendation
7. Do not make design decisions without rationale and evidence links.
8. Update decision log whenever a strategic choice is made.
9. Reuse existing context packs before creating new ones.
10. Keep outputs concise but auditable.
