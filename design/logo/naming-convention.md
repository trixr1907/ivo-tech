# Naming Convention (Logo Assets)

## Rules
- Brand spelling in content and metadata: `ivo-tech`
- File names: lowercase, hyphen-separated, deterministic.
- Do not include version numbers in public delivery paths.
- Keep format suffix explicit (`.png`, `.webp`, `.avif`, `.mp4`, `.webm`, `.ico`, `.glb`, `.stl`).

## Canonical Public Paths
- `/assets/logo.png`
- `/assets/logo.webp`
- `/assets/logo.avif`
- `/assets/logo-mark.png`
- `/assets/logo-mark.webp`
- `/assets/logo-mark.avif`
- `/assets/video/logo-sting.mp4`
- `/assets/video/logo-sting.webm`
- `/assets/video/logo-sting-poster.avif`
- `/assets/video/logo-sting-captions.vtt`
- `/favicon.ico`
