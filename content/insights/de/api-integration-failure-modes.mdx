Die teuersten Integrationsfehler sind selten Syntax-Fehler, sondern Annahmen ueber Verhalten externer Systeme.

## Kritische Failure-Modes

- Instabile Rate-Limits ohne Backoff
- Nicht-idempotente Retry-Logik
- Schema-Drift ohne Versionierung
- Fehlende Dead-Letter-Strategien
